# Expense Tracker

A terminal-based expense tracker built with Python using Object-Oriented Programming.

## Features
- Add expenses with category and description
- View all expenses with date
- View total spending
- View spending by category
- Delete expenses
- Data saved to file and loaded automatically on next run

## Concepts Used
- Python OOP (Classes, Objects, Inheritance)
- File handling
- Exception handling
- Dictionaries and Lists

## How to Run
1. Clone the repository
   git clone https://github.com/swarajpatil10/expense-tracker.git
2. Navigate to folder
   cd expense-tracker
3. Run the program
   python expense_tracker.py

## Tech Stack
- Python 3.x
